Microservices Architecture

Технологии: Go, Kubernetes, RabbitMQ

Микросервисная архитектура для обработки больших объемов данных в реальном времени.

Пример API-запроса:


package main

import (
    "fmt"
    "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Привет, это микросервис!")
}

func main() {
    http.HandleFunc("/", handler)
    http.ListenAndServe(":8080", nil)
}

Task Management API

Технологии: Python, Django, PostgreSQL

API для управления задачами с поддержкой авторизации через JWT.

Пример API-запроса:


from rest_framework.views import APIView
from rest_framework.response import Response

class TaskList(APIView):
    def get(self, request):
        tasks = Task.objects.all()
        serializer = TaskSerializer(tasks, many=True)
        return Response(serializer.data)
